# Building Simple Transcriber

## Prerequisites

- [AutoHotkey v2](https://www.autohotkey.com/) installed to `C:\Program Files\AutoHotkey`
- The Ahk2Exe compiler (included with AutoHotkey v2)

## Compilation

### Quick Build

```
scripts\compile.bat
```

This compiles `src/main.ahk` into `build/SimpleTranscriber.exe`.

### Full Build

```
scripts\build-all.bat
```

## Output

- `build/SimpleTranscriber.exe` - Standalone executable
- `build/settings.default.ini` - Default configuration
